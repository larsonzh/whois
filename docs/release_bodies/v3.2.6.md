# whois v3.2.6

本次发布聚焦稳定性与发布体验优化：
- 工作流手动触发的 tag 输入支持自动裁剪与规范化（`3.2.6`/`V3.2.6` → `v3.2.6`）。
- 远程构建脚本 `-s` 支持多路径与 Windows 路径自动规范化，VS Code 任务默认双路径同步。
- 发布正文采用“隐式下载链接”风格，减少视觉噪音。
