# whois v3.2.11

本次版本聚焦 CIDR 权威收敛稳定性、LACNIC→ARIN 规则细化与 DNS 可观测性补强，并完成多轮严格门禁验证。
This release focuses on CIDR authority convergence stability, LACNIC→ARIN rule refinement, and stronger DNS observability with repeated strict gate validation.

中文概要：
- 重大改进（规则契约基线）：本版以《IPv4/IPv6 地址 WHOIS 查询规则契约》作为实现与评审主入口（`docs/RFC-ipv4-ipv6-whois-lookup-rules.md`），后续判定语义以该文档为准。
- CIDR 契约收敛：固化“原始查询 + 基准回查 + 一致性验证 + APNIC 前候选回查”闭环，减少顺序漂移与误收敛。
- LACNIC→ARIN 规则细化：将“立即非权威跳转且不预标记 ARIN visited”触发条件收敛为“非 IP 字面量查询”，IP 字面量保留既有权威路径。
- 开关治理推进：`--no-cidr-erx-recheck` 已进入 deprecated 阶段（保留兼容窗口，运行时一次性告警）。
- 可观测性增强：新增 `[DNS-CAND-IANA]`、`[DNS-CAND-SUM]`、`[DNS-CAND-RATIO]`、`[DNS-CAND-UNIQ]` 调试标签（仅 `--debug`/`--retry-metrics`）。
- 验证基线（2026-02-25）：
  - Strict Version 两轮（`lto-auto`）：默认参数与 `--debug --retry-metrics --dns-cache-stats --dns-family-mode interleave-v4-first` 均 PASS（无告警 + 无 LTO 告警 + Local hash verify PASS + Golden PASS + referral check PASS），日志 `out/artifacts/20260225-202815`、`out/artifacts/20260225-203408`。
  - CIDR Contract Bundle（prefilled）PASS：`result=pass`、`body_status=pass`、`matrix_status=pass`，汇总 `out/artifacts/cidr_bundle/cidr_bundle_summary_20260225-210527.txt`。
  - Redirect Matrix 10x6 PASS：`authMismatchFiles=0`、`errorFiles=0`，目录 `out/artifacts/redirect_matrix_10x6/20260225-204314`。

English summary:
- Major improvement (contract baseline): this release adopts the “IPv4/IPv6 WHOIS lookup rules contract” as the primary implementation/review entry (`docs/RFC-ipv4-ipv6-whois-lookup-rules.md`), and subsequent authority semantics follow that contract.
- CIDR contract convergence: solidifies the closed loop of “original query + baseline recheck + consistency validation + pre-APNIC candidate lookback” to reduce order-dependent drift.
- LACNIC→ARIN refinement: narrows the “immediate non-authoritative continuation with no ARIN pre-visited mark” trigger to non-IP-literal queries; IP literals keep the existing authority path.
- Flag governance: `--no-cidr-erx-recheck` is now deprecated with a compatibility window and one-time runtime warning.
- Observability upgrades: adds `[DNS-CAND-IANA]`, `[DNS-CAND-SUM]`, `[DNS-CAND-RATIO]`, and `[DNS-CAND-UNIQ]` debug tags (only under `--debug`/`--retry-metrics`).
- Validation baseline (2026-02-25):
  - Strict Version, two rounds (`lto-auto`) PASS for both default args and `--debug --retry-metrics --dns-cache-stats --dns-family-mode interleave-v4-first` (no warnings + no LTO warnings + Local hash verify PASS + Golden PASS + referral check PASS), logs `out/artifacts/20260225-202815` and `out/artifacts/20260225-203408`.
  - CIDR Contract Bundle (prefilled) PASS with `result=pass`, `body_status=pass`, `matrix_status=pass`, summary `out/artifacts/cidr_bundle/cidr_bundle_summary_20260225-210527.txt`.
  - Redirect Matrix 10x6 PASS with `authMismatchFiles=0` and `errorFiles=0`, artifact dir `out/artifacts/redirect_matrix_10x6/20260225-204314`.
