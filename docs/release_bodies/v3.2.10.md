# whois v3.2.10

本次版本聚焦 ARIN 前缀剥离策略与回归自测，补齐可观测性与发布基线记录。
This release focuses on ARIN prefix stripping and a deterministic lookup selftest, with refreshed golden baselines.

中文概要：
- ARIN 前缀剥离：当查询带 ARIN 风格前缀但当前 hop 不是 ARIN 时自动剥离，并在 stderr 输出 `[DNS-ARIN] strip-prefix`。
- 新增 lookup 自测 `arin-prefix-strip`：纯字符串校验，无网络依赖，用于守住前缀剥离规则。
- 四轮黄金（2026-01-15）全部 PASS：默认 / debug+metrics / 批量四策略 / 自检四策略；日志 `out/artifacts/20260115-112537`、`20260115-113007`，以及 `batch_{raw,health,plan,planb}/20260115-11{3500,3857,4216,4510}/...` 与 `batch_{raw,health,plan,planb}/20260115-11{5135,5533,5808,0129}/...`。

English summary:
- ARIN prefix stripping: non-ARIN hops now strip ARIN-style prefixes and emit `[DNS-ARIN] strip-prefix` on stderr.
- New lookup selftest `arin-prefix-strip`: pure string validation, no network dependency.
- Four-way golden matrix (2026-01-15) all PASS: default, debug+metrics, batch four strategies, selftest four strategies; logs as above.
